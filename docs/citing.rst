Citing
******

If you use the `radcap`_ code in any form, please cite the following manuscript
(which is "in review"):

.. [SH2016] Hoffberg S, Kieran TJ, Catchen JM, Devault A, Faircloth BC, Mauricio
   R, Glenn RC. 2016. Adapterama IV: Sequence Capture of Dual-digest
   RADseq Libraries with Identifiable Duplicates (RADcap).  bioRxiv.
   doi:`10.1101/044651 <http://doi.org/10.1101/044651>`_.

Please also cite the github repository:

.. [BCF2016] Faircloth BC. RADcap software.
   https://github.com/faircloth-lab/radcap.
